Hand Coding Round Robin

below are 33 pages from two online workshops held on July 28 + 29, 2020 with Melanie Hoff's Digital Love Languages class. led by emma rae norton. 34 participants collaboratively wrote love letters as web pages as love letters with each other. through adding and styling the content of each others web pages students learned the basic building blocks of web development as well as what it feels like to code with and for each other.
