hand coding round robin

a one hour workshop where a group of students will learn how to hand code a web page by working on each others computers

taught by emma rae norton

  • REQUIRED STUFF 2 DO B4 ATTENDING WORKSHOP:
  • you will need to bring your own computer. mentally prepare for the fact that other participants will be using your computer, and you theirs !
  • please download this file and save it to your desktop (don't worry about the code inside we will go over this file in depth).
  • have a text editor installed. atom, sublime text, brackets and visual studio code are all great options. it will be helpful if you are somewhat familiar with opening, writing to, and saving files within this editor.
  • have a web browser installed. chrome and firefox are preferred but not required.
