what is hand coding

all webpages are just files and all of them have a file called the index.html file. this is always the first file that you see when you go to a website. even the really fancy ones.

in this workshop we are going to move away from the bloated websites of 2019 and learn how to build a webpage slowly, by hand, and with care. its possible it will look like a webpage from the 90s but it doesn't have to. the funny thing is that the languages, HTML, CSS, and Javascript are very much the same now as they were back then and its more about your content, how you style it and how you leverage the inherent functionality of these three languages.

1

open the index.html file that you downloaded before this workshop, it should be on your desktop, open it in a text editor like Atom or Sublime. also make sure that you have a webbrowser open in the background so that other students can easily access it when they are one your computer.

lets look at the css at the top of page, line 16

everything underneath the word body is where you'll put the most basic styles for your web page. these can be things like the backgroud color, or maybe a border. you can use padding to add space between the edge of the browser and the content on your page. get souped ! choose pixel values and colors that you feel good about. 

          body {

            border: 10px solid blue;

            background-color: yellowgreen;

            color: mediumpurple;

            font-size: 30px;

          }
