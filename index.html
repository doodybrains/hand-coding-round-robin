1

find the index.html file on your desktop. rename the file to your-name.html

double click the file to open file in a browser. you should also open the file in a text editor. both the browser window and text editor window should be easily accessible so that other students can easily work on the file.

let's look at the file in a text editor together...make a CSS change to the body tag

2

scroll down and place your cursor somewhere inside the body tag

inside of an h1 tag, please write a sentence or phrase responding to this prompt:

think of a metaphor for the world wide web.

make sure to write something that you would want your peers to see

and that its something fun for them respond to !

<h1 id="metaphor">the world wide web is a big plate of spaghetti and meatballs, with extra parmigiano-reggiano</h1>

after you add your metaphor, scroll back up to the CSS section and add some styles to your new element using the metaphor id:

3

scroll down and place your cursor somewhere inside the body tag, a few lines below the h1 tag

p tag inside of an div tag, please write a short response to the h1 tag phrase, think about the subtext of the metaphor:

you should also add an id, the name of the id can be anything you want, refer to the id in your css to add styles <div id="response">
<p>all of the fiber optic cables are like a big mess of spaghetti, facebook, amazon, google, just a bunch of meatballs, but the sprinkling of hand coded websites on top of it all makes it ok</p>
</div>

4

now open a browser window and search for an image in response to something written in the file

copy the image address to your clipboard

back in the editor, scroll down and place your cursor somewhere inside the body tag, a few lines below the response div

create an image tag

paste the url for the image inside of the quotes, after src= attribute

don't forget to add a description of the image inside the alt text attribute!!

<img id="name-of-image-id" alt="description of the image" src="http://url-of-image.com"/>

5

what is the name of the id on the image tag?

scroll up to the css section of the document an create a list of styles for that id

it should look something like this

this css tells your image to take up 20% of the window width, try out some of the styles like the position and transform rules

save your file and open it in a browser to see what it looks like

now you can add more styles to your image 

          #name-of-id {

            width: 20%;

            transform: rotate(-20deg);

          }
